# Security Policy

## Reporting a vulnerability

Email security@cubenest.in. We commit to:
- Acknowledge within 72 hours
- Provide a status update within 7 days
- Coordinate disclosure timing with you

Do not file public GitHub issues for security vulnerabilities.

## Supported versions

Each package's latest minor receives security fixes. Older minors are best-effort. See per-package CHANGELOG.md for current versions.
